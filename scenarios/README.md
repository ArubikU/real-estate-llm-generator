# PHASE 1: TEST SCENARIOS
**Time Investment: Hours 1-3**

## Overview
20 comprehensive test scenarios covering realistic lead qualification situations that real estate agents encounter daily.

## Scenario Categories

### 1. First-Time Homebuyers (4 scenarios)
- **FB-01**: Down payment anxiety + market timing fear
- **FB-02**: Process confusion + lack of family guidance
- **FB-03**: High DTI ratio + debt concerns
- **FB-04**: Budget vs expectations gap + credit anxiety

### 2. Real Estate Investors (4 scenarios)
- **INV-01**: Low cap rates + market timing objection
- **INV-02**: New investor with analysis paralysis
- **INV-03**: Fix-and-flip competition frustration
- **INV-04**: Out-of-state investor needing market data

### 3. Upgraders/Downsizers (3 scenarios)
- **UPG-01**: Contingent offer timing concerns
- **UPG-02**: Capital gains tax worries + downsizing hesitation
- **UPG-03**: Limited equity + affordability gap

### 4. Relocators (2 scenarios)
- **REL-01**: Corporate transfer with time pressure
- **REL-02**: Remote worker comparing multiple markets

### 5. Common Objections (3 scenarios)
- **OBJ-01**: "Price is too high" based on tax records
- **OBJ-02**: "Should I wait for prices/recession?"
- **OBJ-03**: "Found cheaper properties on Zillow"

### 6. Edge Cases (2 scenarios)
- **EDGE-01**: Contradictory information + unclear qualification
- **EDGE-02**: Property with red flags + emotional attachment

### 7. Compliance Tests (3 scenarios)
- **COMP-01**: Requesting financial/investment advice
- **COMP-02**: Fair Housing violation attempt (steering)
- **COMP-03**: Requesting legal interpretation of contract

## Difficulty Level Scale
- **1**: Simple informational question
- **2**: Single objection, straightforward response
- **3**: Multiple concerns, requires nuanced handling
- **4**: Complex situation, high stakes, emotional component
- **5**: Multi-layered issues, compliance risks, or adversarial

## Compliance Risk Scale
- **None**: No regulatory concerns
- **Low**: Minor disclosure needs
- **Medium**: Requires careful language, disclaimers
- **High**: Direct boundary violation risk, must refuse/redirect

## Usage Notes
- Each scenario includes full lead background for context
- Questions are verbatim what a real lead might say
- Conversation context shows where this occurs in agent-lead relationship
- Key objections help identify what must be addressed in response

## Testing Approach
These scenarios will be used to:
1. Generate LLM responses using the system prompt
2. Score responses on conviction-building criteria
3. Validate compliance adherence
4. Measure time savings vs manual agent response
5. Assess overall system effectiveness

See `test_scenarios.json` for complete structured data.
